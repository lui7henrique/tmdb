**Plotwist** は、映画、シリーズ、アニメ、ドラマなどを管理しレビューするための完全なプラットフォームで、Letterboxd や MyAnimeList のような代替案とは異なり、すべてを1か所で行えます。

#### **無料プラン**
- 無制限のレビューとリスト。  
- 視聴済み、視聴中、視聴予定のコンテンツを管理するツール。  
- 共同リストおよび友達へのおすすめ機能。  

#### **プロプラン**
- 詳細な統計 (視聴時間、ジャンル、出演俳優など)。  
- 個人履歴に基づくおすすめ。  
- バッジと成果。  
- 新機能への優先アクセス。  

#### **目的**
活気あるコミュニティを構築し、視聴型エンターテインメント愛好者のための究極の目的地になること。
