# プロットウィストについて

**Plotwist** は、すべてを1つのプラットフォームにまとめるために作られました。映画だけに焦点を当てた Letterboxd などのプラットフォームとは異なり、Plotwist では映画、シリーズ、アニメ、ドラマなどを管理してレビューすることができます。

私たちの目標はシンプルです。あなたのリストを整理し、観たものを追跡し、次に観たいものを計画するための、完全で洗練されたスペースを提供することです。

## 機能

- **整理と管理**  
  観たものやこれから観る予定のものを簡単に追跡。

- **パーソナライズされたおすすめ**  
  あなたの好みや視聴履歴に基づいたおすすめを受け取る。

- **無制限のリスト**  
  必要なだけリストを作成可能。

- **高度なカスタマイズ**  
  独自の好みに合わせて体験をカスタマイズ。

- **実績とバッジ**  
  報酬を解除し、楽しい実績で進捗をアピール。

- **TMDB統合**  
  映画やシリーズなどの豊富なカタログに完全な情報でアクセス。
